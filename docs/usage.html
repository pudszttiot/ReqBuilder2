How To Use

Follow the steps below to get started with ReqBuilder 2. Whether you're a beginner or an experienced user, these instructions will help you use the app to its full potential.

Step-by-Step Instructions

    Step 1: Before using ReqBuilder 2, ensure that it is installed correctly by following the Installation Guide.

    Step 2: Run the tool using the command 'python ReqBuilder2.py'

    Step 3: Click the "Select Project Directory" button to choose your project directory.

    Step 4: (Optional) Enter a custom output path for the 'requirements.txt' file.

    Step 5: Once the project directory is selected, click the "Generate requirements.txt" button to create the 'requirements.txt' file.

    Step 6: The output log box will show success or errors.

    Step 7: (Optional) Click the "Clear Output" button to reset the status area and clear all logs.

    Step 8: (Optional) Click the "Check for Updates" button to verify if a newer version of ReqBuilder 2 is available or to ensure youâ€™re on the latest version.
